## Problem 3 – Sum Each Row in Matrix

### Description
Write a program to fill a 3×3 matrix with random numbers, then calculate the sum of each row and store the results in a separate array.

### Requirements
- Generate a 3×3 matrix with random numbers from 1 to 100
- Calculate the sum of each row
- Store row sums in a separate array
- Print the matrix and the sum of each row

### Concepts Used
- 2D Arrays
- Functions
- Loops
- Random number generation
- Passing arrays to functions

### Language
- C++
