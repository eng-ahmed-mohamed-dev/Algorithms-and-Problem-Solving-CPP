## Problem Description
Write a program to print the Fibonacci series of 10 numbers using recursion.

The Fibonacci series is a sequence where each number is the sum of the two previous numbers.

## Solution Idea
The solution uses a recursive function that keeps track of the previous two numbers.
In each recursive call, a new Fibonacci number is calculated, printed, and passed to the next call.
The recursion continues until the required count is reached.
