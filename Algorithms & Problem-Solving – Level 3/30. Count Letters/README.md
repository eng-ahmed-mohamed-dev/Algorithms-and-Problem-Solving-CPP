# Problem 30 – Level 3 | Count Letters

## Problem Description
Write a program that reads a string and a character from the user, then counts how many times the given character appears in the string.

## Solution Idea
- Read a full string from the user.
- Read a single character.
- Loop through the string and compare each character with the given one.
- Increase a counter every time a match is found.
- Print the final count.

## Used Concepts
- Strings
- Loops
- Functions
- Character comparison
- Input / Output

## Files
- `Count Letters.cpp` → C++ solution

## Notes
- The comparison is case-sensitive.
- Spaces inside the string are counted normally.
