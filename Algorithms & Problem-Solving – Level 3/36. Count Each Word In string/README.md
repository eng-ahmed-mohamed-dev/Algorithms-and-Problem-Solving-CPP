## Problem Description
Write a program that reads a string from the user and counts how many words it contains.
A word is considered any sequence of characters separated by spaces.

## Solution Idea
- Read the full string using `getline`.
- Split the string using space as a delimiter.
- Count only valid words and ignore extra spaces.
- Return the total number of words found.

## Used Concepts
- String manipulation
- Functions
- Loops
- `getline`
- `find` and `erase`
