Check Matrices Equality

## Problem Description
Write a program to generate two 3x3 matrices with random numbers, then check if they are equal or not.

The equality check in this problem is based on comparing the **sum of elements** in each matrix.

## Solution Idea
- Generate two matrices with random values.
- Print both matrices.
- Calculate the sum of elements for each matrix.
- Compare the two sums.
- If the sums are equal, the matrices are considered equal.
