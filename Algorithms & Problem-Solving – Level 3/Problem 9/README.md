Print Middle Row and Column of a Matrix

## Problem Description
Write a program that fills a 3x3 matrix with random numbers, prints the matrix, then prints the middle row and the middle column.

## Solution Idea
- Create a 3x3 matrix.
- Fill the matrix with random numbers between 1 and 10.
- Print the whole matrix.
- Calculate the middle row and middle column using integer division.
- Print the middle row and then the middle column.

- 2D Arrays
- Functions
- Loops
- Random number generation
- Formatting output
