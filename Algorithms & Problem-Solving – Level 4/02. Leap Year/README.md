# Problem 02 – Leap Year (Level 4)

## Problem Description
Write a program to check whether a given year is a leap year or not.

A year is considered a leap year if:
- It is divisible by 400, or
- It is divisible by 4 but not divisible by 100.

## Solution Idea
The program reads a year from the user, then applies the leap year rules using a boolean function.
The function returns true if the year satisfies the leap year conditions, otherwise false.

## Used Concepts
- Functions
- Boolean logic
- Modulus operator (%)
- Conditional statements
- Console input/output
